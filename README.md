# Amiga Green Tech

React-based About Us site for **Amiga Green Tech** — renewable energy, solar power, EV batteries, energy storage, and sustainable power solutions.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build for production

```bash
npm run build
npm run preview
```

## Stack

- **React 18** + **Vite**
- Standard CSS (no UI framework)
- Business-oriented layout and typography (DM Sans, Fraunces)

## Content

The About Us page includes the company’s six main objectives: renewable energy technologies, R&D, energy storage & smart systems, consultancy & EPC, collaboration & partnerships, and intellectual property.
